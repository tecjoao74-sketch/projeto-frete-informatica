# Projeto Frete Informática

Sistema de cálculo de frete usando API ViaCEP.

## Funcionalidades
- Consulta de CEP usando API ViaCEP
- Identificação de cidade e estado
- Cálculo de frete baseado na localização

## Tecnologias usadas
- HTML
- JavaScript
- API ViaCEP

## Como executar

1. Instale a biblioteca requests:

pip install requests

2. Execute o programa:

python frete_pe.py
