# Projeto Frete Informática

Sistema de cálculo de frete usando API ViaCEP.

## Funcionalidades
- Consulta de CEP usando API ViaCEP
- Identificação de cidade e estado
- Cálculo de frete baseado na localização

## Tecnologias usadas
- HTML
- JavaScript
- API ViaCEP

## Como executar

1. Baixe o projeto ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Digite um CEP válido e veja o cálculo do frete.
